Easm 可以理解为 “Easy Assembly”，它是一个专注于动态机器码生成、运行时指令构建以及字节级汇编流程的现代 C++ 项目。这个项目的目标，是提供一套清晰、富有表达力且可控的基础能力，用来支撑运行时生成可执行代码的需求，并逐步形成属于 Easm 自己的接口风格与工程方向。
